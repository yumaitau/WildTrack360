# **App Name**: WildTrack360

## Core Features:

- Animal Record Timeline: Display animals' details and chronological record histories. Each record consists of Growth measurements, Health checks and medical notes, Diet & feeding history, Carer assignments, Release details & post-release monitoring.
- Visual Documentation: Support photo uploads for each record (especially to show injuries or development stages) and display a carousel/gallery with latest image highlighted.
- Advanced Record Search: Enable users to find records by species, status, date range, or carer.
- Record Export: Allows single record CSV export, as well as the mass export of records.
- Image Autotagging: Utilize a tool to intelligently parse images from users to provide an automatically generated brief description for logging or archival purposes.

## Style Guidelines:

- Primary color: Forest Green (#3F704F) to evoke a sense of nature and wildlife.
- Background color: Light Beige (#F5F5DC), a desaturated tint of forest green that is calm and natural, providing a neutral backdrop.
- Accent color: Earthy Brown (#8B4513), an analogous color that provides contrast and complements the natural theme.
- Font pairing: 'Alegreya', a humanist serif (headlines), with 'PT Sans' (body text)
- Use nature-inspired icons related to animal species and care activities.
- Clean, card-based layouts for animal records to ensure easy navigation.
- Smooth transitions and subtle animations for loading data and displaying images.
